1.面向对象相关概念
1.1 js 是不是一个面向对象的语言?
        不是: 与传统面向对象的理论有矛盾.（C#, Java ）
        是: js 里面到处都是对象. 数组, 时间, 正则 ... 和 DOM
            也可以像传统面向对象的语言那样用 new 的方式创建对象
1.2 js 是一个基于对象的多范式的编程语言

        多范式：就是编程风格（可以有多种编程风格）
            可以使用的编程方式：a.面向过程的方式编程
                                             b.面向对象的方式编程          
                                             c. 函数式等
            --->扩展：函数式的一个特点: 递归与链式
                               jQuery 就是典型的链式编程风格
                                比如: 给 div 标签添加样式

$( 'div' ).css( 'border', '1px solid red' )
          .css( 'width', '400px' )
          .css( 'height', '200px' );
1.3 面向对象和面向过程的区别
        1.3.1 面向过程-->关注细节
                    面向过程开发 -> 使用过程的方式开发
        1.3.2 面向对象-->直接找人做事，一步到位
                     面向对象开发 -> 使用对象开发
                      要做什么, 就找到对应的对象, 告诉它做, 等结果
             -> 是否说面向对象可以取代面向过程
                  当然不可以！
        总结： 面向对象是面向过程的封装（简单粗暴，一步到位）
1.4  什么是对象
     对象可以理解为数据集或功能集.
        -->数据集(打包的数据集合)：
               eg:{ name: '张三', age: 19, gender: '男' },json值等
        -> 功能集(函数集, 方法集)：
               将很多个方法打包，多次调用
               -> 数据集和功能集也可以混合使用
            例如 jQuery:
                $( 'div' )    这个是一个 jq 对象, 但是实际上里面包含 DOM 对象
                $( 'div' ).css( ... )
                 $( 'div' ).html( ... )    是不是就说明 jq 对象中包含方法
在 js 中, 所谓的对象就是 键值对的 集合

2.3.1调试按钮的功能
     a.逐语句: 单步运行. 就是一次只执行一句话. 如果遇到函数, 进入函数体内一步一步执行
     b.逐过程: 一次执行一个过程. 如果是普通语句那么与逐语句是一样的. 但是如果是一个函数, 那么他就会将函数执行完, 立即跳到下一步
     c. 继续运行 ：从当前断点开始继续运行，直到碰到下一个断点停止，如果一直没有断点，则执行完毕

3.JS的数据类型及存储模型
     3.1数据类型
  • 基本类型( 值类型 ):直接存储值（存储在栈）
          数字 number, 字符串 string, 布尔 boolean
  • 复合类型( 引用类型 ):也就是复杂数据类型---存储引用（地址在栈，栈中的地址指向堆中的内容）
           对象( 数组, 时间类型, 函数类型, 正则表达式, ... )
  • 空类型: Null, Undefined（存储在栈）
     3.2存储模型
     基本数据类型和空类型的存储模型---一个方格里面放一个数据（就是所谓的栈中存储）
     复合类型的存储模型---独立的存储区域，（栈中存地址，地址指向堆中存储的内容）
4.数据类型的判断
        复合类型, 除了函数, 其他的数据类型无法使用 typeof 获得数据类型名（无法获取到具体的数据类型）
    注意: 如果需要获得对象的数据类型, 需要使用                      Object.prototype.toString.apply( 数据 )
      或者写成：Object.prototype.toString.call( 数据 )

var arr=[1,2,3,4];
var sum= function () {
    var sum = 0;
    for (var i = 0; i < 6; i++) {
        sum = sum + i;
    }
    console.log(sum);
}
console.log(typeof arr);//object
console.log(Object.prototype.toString.apply(arr));//[object Array]
console.log(typeof sum);//function
